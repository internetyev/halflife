# halflife

A viral web tool that tells you, given a job title, how long it has before AI replaces ≥50% of its tasks — with a survival score, the AI tools already disrupting it, and a personalised pivot roadmap.

Status: planning. The build is driven by an autonomous Claude Code routine that runs at 03:00 Europe/Madrid each day.
